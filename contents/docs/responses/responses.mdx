# Responses

1. **Can I filter responses by a specific form?**

   Yes, you can filter responses to see submissions from a specific form or view responses from all forms.

2. **Is it possible to search through the responses?**

   Yes, you can use the search bar to search for specific responses based on keywords or details within the submissions.

3. **How do I view only responses from one particular form?**

   Select the desired form from the filter dropdown to display responses for that specific form only.

4. **What should I do if I have many responses and want to find a certain one quickly?**

   Use the search bar at the top of the Responses page to enter relevant keywords, and view only the submissions that match your search.
